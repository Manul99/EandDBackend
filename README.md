# Employee And Department Backend

## Download the database and restore it from this
````
https://drive.google.com/file/d/1sqGiXBWZE0x1K_XnWTzgbzs47LsSaoFl/view?usp=sharing
````
## Clone the project
````
git clone https://github.com/Manul99/EandDBackend.git
````
## Add the server name 
````
  "ConnectionStrings": {
    "DefaultConnection": "server=Your_server; database=EmployeesAndDepartmentsDB; Trusted_connection=True; TrustServerCertificate=True;"
  },
````
